Template Variable Define Tag 

=>

The template variable define tag allows you to set 'variables' whose values will be pasted into your docuement's template. (More on creating your own template later)

Example: 

=.author>your_name

To place a template value in your document, you use the pseudo-HTML tag, '<swl>'

Example: 

<swl author>

yields: your_name.

Builtin Templates

The SWL compiler automatically sets these templates so that you can insert them into your document:

date date of compilation
author the default author is "SWL version"
body if you define a 'template' variable, the content of your document will be dumped into the 'body' template so you can place it wherever you want it.

about.swlt Templates

If you put the line '+>about' on the top of your document, these template variables are set:

  • swl.author
  • swl.version
  • swl.release date
  • swl.copyright
  • swl.license

Note that when you compile a SWL document, SWL tries to find local.swlt in the same directory as your source code or one of the parent directories. Thusly, you can port generic SWL files from one site to another, recompile, and completely change the look of your pages.
