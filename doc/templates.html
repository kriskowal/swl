Templates

This is where most of the power of SWL is. There will be a quiz.

Templates... Templates have 3 aspects:

  • the template, the framework for you HTML document, defined using '=.template>'
  • template tag, for placement of values defined in your document, using '<swl your variable>'
  • template definition, using '=.your variable' in your document
